Welcome to www.php-fig.org.

This is just some placeholder content until the domain is in place, we'll get real meat of the project added as we go.

Here is a little quickstart to hack on _this_ website. Edit as needed for your dev style

$ mkdir -p ~Projects/web/www.php-fig.org && cd ~Projects/web/www.php-fig.org
$ git clone git@github.com:php-fig/php-fig.github.com.git
$ git checkout gh-pages

Or If you're using the GitHub for Mac, simply sync your repository and you'll see the new branch.

Then when finish, just commit your changes and push them or submit a pull request, what ever makes more sense according to your changes.

Ready to rock and roll!
